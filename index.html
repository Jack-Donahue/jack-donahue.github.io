Space Fighters

I made a Galaga style space shooter game using Unity. I made a player movement system. enemy spawner system, and a score system. The greatest obstacle I faced while making this game was the score system and its relationship with how frequently enemies spawned. I solved this by making a singleton that could hold the value of the state of the game.

Guess the Food!

This is my introduction into 3D game development using Unity. I created a movement system, a dialogue system, and a quiz system. I was able to create the quiz and movements systems quite easily. However, the dialogue took a lot of time to get used to. I eventually got around this be realizing I had to make certain parts of the game assigned to different events so I could separate the dialogue from the actual gameplay mechanics.
